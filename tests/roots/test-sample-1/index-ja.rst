このモジュールが作られたきっかけは、私がPythonでMS Word .doc ファイルを
扱えないかPyPIやStackoverflowで探したことだ。残念ながら見つけることが
出来たのは以下の方法だけだった:

1. COM 操作
2. .net か Java
3. OpenOffice か MS Office の操作

私は、もっとシンプルなものが欲しかったので docx を作った。

ドキュメントの作成
===================

docx モジュールには以下の機能がある:

* パラグラフ
* 箇条書きリスト
* 番号付きリスト
* 複数レベルの見出し
* テーブル
* ドキュメントのプロパティ

テーブルについては、以下のようなリストのリストのみ対応している:

== == ==
A1 A2 A3
B1 B2 B3
C1 C2 C3
== == ==
